## 📡 Zdroje dat

### Primární (ověřené, strojově čitelné)

| Zdroj | Použití | Dostupnost |
|-------|---------|-----------|
| **Deezer API** | Tracklisty, alba, metadata, duration, explicit tagy | REST, free tier (omezený), public endpoints pro artist/album/track |
| **Spotify Web API** | Stejné jako Deezer + popularity, monthly listeners, related artists | REST, vyžaduje OAuth client, rate-limited |
| **YouTube Data API** | Oficiální videa, play count, kanály, playlisty | REST, quota 10k/den, API key zdarma |
| **Wikipedia (CS/SK/EN)** | Biografie, diskografie, historie — structured infoboxy | REST (action=parse), CC-BY-SA, bez rate limitu |
| **Discogs API** | Label info, vydání, katalogy, mastering | REST, 25 req/min, API key zdarma |
| **Genius API** | Lyrics, anotace, významy, storytelling | REST, API key zdarma |

### Automatizační tipy

- **Deezer API**: `https://api.deezer.com/artist/{id}/albums` + přímé URL podle `deezerTrackId` v datech
- **Spotify**: Použij `spotify-web-api-node` npm balík pro Node.js skripty
- **YouTube**: Jednoduché scrapery v Pythonu přes `youtube_dl` nebo `yt-dlp` metadata
- **Wikipedia**: API call na `https://cs.wikipedia.org/w/api.php?action=parse&page=Název&format=json&prop=text`
- **Genius**: API search pro texty — `https://api.genius.com/search?q=artist+song` + extract lyrics z HTML
